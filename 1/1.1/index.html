    1.primitive type

    1. 基本内置类型

    基本类型：bool, char, short, int, long, long long, float, double, long double
    扩展类型：
    wchar_t： 宽字符，16bit，char 保证可以存放机器基本字符集的字符，wchar_t 保证可以存放机器最大扩展字符集中的字符。
    char16_t、char32_t：分别为 16bit、32bit，为 Unicode 字符集服务。

    类型使用建议：

    • 明确数值不为负，使用无符号类型；
    • 算术表达式不要使用 char 和 bool，只在存放值时使用它们。因为 char 是否有符号视机器而定，如果需要小整数运算，则最好指明 signed char/unsigned char；
    • 浮点数运算使用 double，float 可能大小不够且速度更慢。

    3. 字面值 (literal)

    整数： 

    1
    20 /* dec */   024 /* oct */   0x14 /* hex */
    • 十进制字面值类型是 int、long、long long 中能容纳且尺寸最小的那个；
    • 八进制和十六进制是int、uint、long、ulong、longlong、ulonglong 中能容纳且尺寸最小的那个；
    • 严格来说，十进制字面量不为负，负值的形式的含义是对字面量做负运算。
    浮点： 
    3.14159   3.14159E0   0.   0e0   .001
     浮点字面值的默认类型为 double。

    字符：
