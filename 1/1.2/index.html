    2.variable

    1. 变量初始化

    在 C++ 中，初始化是一个复杂的问题，初始化和赋值是两个不同的操作，但在很多其他的语言中二者的区别可以忽略不计。

    • 初始化：创建变量时赋予一个初始值；
    • 赋值：把对象的当前值擦除，并以新值替代。

    (1). 列表初始化 （list initialization)

    可通过以下四种方式初始化一个变量： 

    1
2
3
4
    int temp = 0;
int temp = {0};
int temp{0};
int temp(0);
    通过花括号初始化即列表初始化，是 C++ 11 的新特性。其重要特点：当初始值存在丢失的风险，编译器会警告。 
    1
2
3
    long double id = 3.1415926536;
int a{id}, b = {id}; // warning
int c(id), d = id;   // correct

    (2). 默认初始化

    定义在函数体外的变量默认值为 0，函数体内部的内置类型如果违背初始化，其值如果被访问将会出错。

    2. 复合类型

     复合类型 (compound type) 是指基于其他类型定义的类型，实现对其他类型的间接访问。C++ 中有多种复合类型。在这里只介绍指针和左值引用。
     <<C++ Primer>> 对 对象 的定义：对象是指一块能存储数据并具有某种类型的内存空间。

    (1). 指针

     void* 是一种特殊的指针类型，可以存放任意对象的地址。从 void* 的视角来看，内存空间仅仅是内存空间，如何访问还需要具体的类型转换。C++ 是强类型的语言，它的类型检查更严格，不允许 C 中支持的 void* 的隐式类型转换。

    1
2
3
4
    // three ways to generate empty pointer in C++
int *p1 = nullptr;
int *p2 = 0;
int *p3 = NULL;
     在 C 语言中，NULL 被定义为 #define NULL ((void*)0)；在 C++ 的 cstdlib 中，出于强类型的原因，它被定义为 #define NULL 0。
     nullptr 是 C++ 11 引入的新特性，应尽可能使用它初始化空指针，nullptr 可以自动转换为任意的指针类型，而 0 既是数字又可以是指针，在函数重载时可能会造成歧义。

    (2). 引用

    引用可以视为对指针的包装，有以下几个特性：

    • 引用必须被初始化，即与一个对象绑定，相当于对象的一个别名；
    • 绑定关系不可修改；
    • 引用只能与对象绑定，不能是字面值或表达式；
    • 对引用的赋值和访问，相当于对原对象的赋值和访问，而不是对绑定关系的修改；
    • 引用本身不是对象，不能生成引用的引用。

    例子： 

    1
2
3
4
5
6
7
8
9
    /* initialize */
int a = 4;
int &b = a;
int &c = b;
/* referencing a pointer */
int i = 1024;
int *p;
int *&r = p;
r = &i;

    3. const 限定符

    即限定一个变量初始化后不可被改变。如： 

    const int BufSize = 512;
    在编译时，编译器会把所有用到 BufSize 的地方都替换为相应的值。如果需要在其他文件中使用，则需要在 const 前加 extern。

    (1). const 引用

    ① const 引用也可能绑定一个非 const 的对象，含义为原对象的值不可通过此引用修改。
     非 const 的引用不能绑定一个 const 对象。

    ② 由于一般的引用可以对原对象进行修改，因此引用的类型要与原对象类型一致，且不能绑定字面量和表达式。  const 引用没有此限制。 

    1
2
3
4
5
6
7
    // case1
int i = 42;
const int &i1 = 42;
const int &i2 = r1 * 2;
// case2
double val = 3.14;
const int &v1 = val;
    ③ 如果 const 引用绑定的类型与引用的类型不一致，编译器会通过隐式转换生成一个临时量。 
    1
2
    const int temp = val;
const int &v1 = temp;

    (2). const 与指针

    $\ast$ 在 const 之后代表这是一个指向常量的指针，const 对象只能被 const 的指针指向。 

    1
2
    const int i = 1024;
const int *i1 = &i;   // <==> int const *i1 = &i;
    $\ast$ 在 const 之前代表这是一个常量的指针。 
    1
2
    int i = 1024;
const int *const comb = &i;

    (3). 顶层 const

     顶层 const (top-level const) 表示指针本身是常量或某个对象是常量，而底层 const 表示指针所指的对象是常量。所有的 const 引用都是底层 const。 

    1
2
3
4
5
6
    int i = 0;
int *const p1 = &i; // top-level
const int c = 42;   // top-level
const int *p2 = &c; // low-level
const int *const p3 = p2; // top + low
const int &r = i;   // low-level
    某个对象被拷贝时，顶层 const 没有什么问题，底层 const 需要考虑其性质是否丢失。 
    1
2
3
4
5
6
    int *px = p1; 
int *py = p3; // ERROR
p2 = p3; 
p2 = &i; // non-const to const
int &r1 = c; // lose lower-const property of referencing c
const int &r2 = i;

    (4). 常量表达式

    常量表达式指值不会改变并且在编译过程中就能确定结果的表达式，比 const 定义更强。字面量、用常量表达式初始化的 const 对象都是常量表达式。 

    1
2
3
    const int max_files = 20;        // const exp
const int limit = max_files + 1; // const exp
const int size = get_size();     // non-const exp
    在复杂的系统中，常量表达式是难以分辨的。C++11 可以将变量声明为 constexpr，实现对编译时常量的静态检查。
     constexpr 限制可声明的类型为字面值类型 (literal type)，包括算术类型、指针和引用，不包括自定义类和 string 类等。
     对于通过函数初始化的 const 变量，如果函数是一个 constexpr 函数 (函数的形参和返回值都是字面值类型，且函数的 return 语句只有一个)，并且函数的实参都是常量表达式，则相应的变量也是常量表达式。

    1
2
3
4
5
6
7
8
9
    constexpr int scale(int arg) {
    return arg * 2;
}

int main() {
    constexpr int val1 = scale(1);
    int i = 100;
    constexpr int val2 = scale(i); // ERROR
}

    对于指针来说，需要注意：

    • 函数体内定义的变量一般来说放在非固定地址中，constexpr 指针不能指向这样的变量；
    • 在 constexpr 声明后定义了一个指针，则只是生成了一个顶层 const，即常量指针。
    1
2
3
4
5
6
    constexpr int i = 42;
int j = 41;
// i and j must be defined out of function
constexpr const int *p1 = &i;
constexpr int *p1 = &j; 
// ERROR: constexpr int *p1 = &i

    4. 附: 声明符的语义分析

    C/C++ 中声明符 (declarator) 的语义规则如下:

    • 声明符成分的优先级: 括号 $>$ 后缀符 ()、[] $>$ 前缀符 (*、&、const、类型等)
    • 前缀符的分析顺序是从右向左 (自底向上)
    • 从标识符 idenfier 开始，按照上述优先级分析语义

    例子: 

    int *&r = p;
// r 是一个引用，引用的是一个指针
char* const (* farr[10])(int);
// farr 是一个数组，数组存放指针，存放的指针指向函数，函数返回 char* const
char * const * (*next)();
// next 是一个指针，这个指针指向一个函数，函数返回一个指针，
//    返回的指针指向 char * const 类型
int (*func(int i))[][20];
// func 是一个函数，函数返回一个指针，指针指向 int[][20] 类型的数组
