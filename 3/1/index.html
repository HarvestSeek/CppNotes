    1.iterator

    1. begin 和 end

    在 C++ 中，和其他语言不同的是，迭代器是容器内部的成员。其基本用法为： 

    1
2
3
    string s("hello world");
for (auto it = s.begin(); it != s.end(); ++it) 
    *it = toupper(*it);

    一般来说，迭代器的类型不被在意。实际上，begin() 和 end() 返回的具体类型由对象是否是常量决定。

    return type of iterator (<TypeName>::%s)
    begin(), end() iterator / const_iterator
    cbegin(), cend()/td> const_iterator
    rbegin(), rend() reverse_iterator / const_reverse_iterator
    crbegin(), crend() const_reverse_iterator

    2. 迭代器操作

    operators of iterator in std containers
    *iter 返回迭代器 iter 所指元素的引用
    iter->mem 解引用 iter 并获取该元素名为 mem 的成员，等价于 (*iter).mem
    ++iter / --iter 移动
    iter1 == iter2 两迭代器指向同一个元素 或 它们是同一个容器的 off-the-end iterator，则相等

    • C++ 中常使用 != 判断循环条件，因为它兼容大多数标准库类；
    • 容器的某些增删元素的操作可能会使迭代器失效。

    3. 迭代器运算

    string 和 vector 的迭代器支持更多的操作符，这些运算被称为迭代器运算 (iterator arithmetic)。

    operators of iterator in std containers
    iter + n / iter - n 移动
    iter += n / iter -= n 复合赋值
    iter1 - iter2 两迭代器之间的距离，参与运算的两个迭代器必须属于同一个容器
    >, >=, <, <= 参与运算的两个迭代器必须属于同一个容器
    其中两迭代器相减得到的类型是 different_type，是一个有符号整数，在 string 和 vector 中均被定义。
