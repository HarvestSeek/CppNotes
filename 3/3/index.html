    3.std::string

    1. 定义与初始化

    ways of initializing a string
    string s1; 默认初始化，s1 是一个空串
    string s2(s1); s2 是 s1 的一个副本
    string s2 = s1; 等价于 s2(s1)
    string s3("value"); s3 是除了 "value" 最后的空字符外的副本
    string s3 = "value"; 等价于 s3("value")
    string s4(n, 'c'); 连续 n 个字符 'c'

    通过等号进行的初始化为拷贝初始化，编译器把等号右侧的值拷贝到新创建的对象中。如果不使用等号，则执行的是直接初始化。

    2. 操作

    operations of string
    os << s 将 s 写入输出流，返回 os
    is << s 从 is 中读取字符串赋给 s，字符串以空白分隔，返回 is
    getline(is, s) 从 is 中读取一行赋给 s，返回 is
    s.empty() true/false
    s[n] 返回 s 中第 n 个字符的引用
    s1 + s2 连接
    s1 = s2 用 s2 的副本代替 s1 中的内容
    s1 == s2 保存的字符完全一样，则相等；大小写敏感
    >, >=, <, <= (第一对相异字符) 字典序比较，大小写敏感

    (2). string::size_type

     string 等标准库都定义了配套的类型，体现了机器无关的特性。作为 s.size() 的返回类型，string::size_type 是一个无符号类型且足够放下任何 string 对象的大小。在 C++11 后，可以通过 auto 和 decltype 进行类型推断。

    3. 处理字符

    在 cctype 头文件中定义了一组处理字符的函数：

    functions in cctype
    isalnum(c) true: c 是字母或数字
    isalpha(c) true: c 是字母
    iscntrl(c) true: c 是控制字符
    isdigit(c) true: c 是数字
    isgraph(c) true: c 不是空格但可打印
    islower(c) / issuper(c) true: c 是小/大写字母
    isprint(c) true: c 是可打印字符 (c 是空格或 c 具有可视化形式)
    ispunct(c) true: c 是标点符号
    isspace(c) true: c 是空格
    isxdigit(c) true: c 是十六进制数字
    tolower(c) / toupper(c) 转换
