    4.basic operations of container

    1. 容器的定义和初始化

    initialization
    C c; 默认构造函数。如果 C 是 array，则按 c 中元素的默认方式初始化，否则 c 为空。
    C c1(c2);
    C c1 = c2;    		 c1 和 c2 必须是相同类型 (容器类型、元素类型)。对于 array，两者大小还必须相同。
    C c{a, b, c...};
    C c = {a, b, c...};    		 列表中元素必须与 C 相容。
    C c(b, e); c 初始化为迭代器指定范围内元素的拷贝。范围中元素类型必须与 C 相容。 array 不适用。
    C seq(n); n 为大小参数。如果没有默认构造器，则还需要传入一个构造器。array、string 不适用。关联容器不支持。
    C seq(n, t); n 个值为 t 的元素。array 不适用。

    通过迭代器指定范围的初始化时，内部类型相同即可。

    vector<const char*> articles = {"a", "an", "the"};

    2. 容器的赋值和 swap

    assignment operation
    c1 = c2; 将 c1 中的元素替换为 c2 中的元素的拷贝。c1 和 c2 的类型必须相同。
    c = {a, b, c...}; 将 c1 替换为列表中元素的拷贝，array 不适用。
    swap(c1, c2);
    c1.swap(c2);    		 交换 c1 和 c2 中的元素，两者类型必须相同，通常比拷贝快得多。
    seq.assign(b, e); 将 seq 中的元素替换为迭代器所指范围内的元素。b 和 e 不能指向 seq 中的元素。关联容器和 array 不支持 assign()。
    seq.assign(i1); 将 seq 中的元素替换为初始化列表 i1 的元素。
    seq.assign(n, t); 将 seq 中的元素替换为 n 个值为 t 的元素。
     赋值运算符要求左边和右边的运算对象类型相同。顺序容器额外支持 assign 操作，允许从一个不同但相容的类型赋值。
     普通容器的 swap 只是交换了容器内的数据结构，数据本身不变，因此速度很快。而对于 array，swap 会真正交换两个容器的元素。新标准提供了非成员版本的 swap()，它对于泛型编程是很有用的。

    3. 容器的大小操作

    size operation
    c.size() 大小
    c.max_size() 返回一个大于或等于该类型容器所能容纳的最大元素的值
    c.empty() size() == 0

    4. 关系运算符

    容器的比较是元素的逐对比较。

    relational operators
    ==, != 所有容器都支持
    >, >=, <, <= 除了无序容器外，所有容器都支持
