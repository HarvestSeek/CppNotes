    1.cstdio

    1. 格式化输入输出

    (1). printf()

    格式化
    %a, %A 浮点数、十六进制数和 p 记数法
    %c 单个字符 %d, %i 有符号十进制整数
    %e, %E 浮点数，e 记数法 %f 浮点数，十进制记数法
    %g, %G 根据值的不同，自动选择 %e 或 %f。小数部分小于 10-4 时选择 %e
    %o 无符号八进制 %x, %X 无符号十六进制
    %u 无符号十进制 %p 指针
    %s 字符串 %% 百分号

    修饰符
    - 打印时左对齐 (默认右对齐) %-10d
    + 显示正负号 -
    空格 显示空格和负号 -
    # 结果转换形式。如八进制填 0、十六进制填 0x、小数保证有小数点 %#X
    数字 指定最小字段宽度，若实际超过此值，系统会使用更宽的字段 -
    .数字 %e, %E, %f: 小数位数
    %g, %G: 有效数字位数 %s: 待打印字符最大数量 整型：待打印数字最小位数，达不到此值的部分填 0    		 %5.2f
    h short int, unsigned short int %hu, %hd
    hh signed char, unsigned char %hhx, %hhd
    l long int, unsigned long int %ld, %lu
    ll long long int, unsigned long long int %lld
    L long double %Ld
    LL 指针差值 ptrdiff_t %td
    z size_t %zd

    (2). scanf()

    格式化
    %a, %A, %e, %E, %f, %F, %g, %G 小数
    其余同 printf()

    修饰符
    * 抑制输入 %*d
    数字 指定最大字段宽度，输入在遇到空格或达到最大宽度后隔断。 -
    hh signed char, unsigned char -
    ll long long, unsigned long long -
    h, l, L %hd, %hi: short int
    %ho, %hx, %hu: unsigned short int
    %ld, %li: long
    %lo, %lx, %lu: unsigned long
    %le, %lf, %lg: double, long double 则把 l 替换为 L
    t ptrdiff_t -
    z size_t -

    (3). * 修饰符

    printf() 和 scanf() 的 * 修饰符的用法不同。

    printf(): 通过程序指定字段宽度，需要额外参数。 

    1
    printf("Weight = %*.*f\n", width, precision, weight);
    scanf(): 跳过相应的输入项。 
    1
2
3
4
    int n;
printf("Please enter three integers:\n");
scanf("%*d %*d %d", &n);
printf("The last integer was %d\n", n);

    2. 文件 IO

    (1). fopen()

    函数原型为 FILE *fopen(const char *filename, const char *mode)

    mode of fopen()
    "r" 读模式
    "w" 写模式，把现有文件长度截为 0，若文件不存在则创建
    "a" 写模式，在现有文件末尾添加内容，若文件不存在则创建
    "r+" 读写模式
    "w+" 读写模式，若文件存在则截为 0，若不存在则创建
    "a+" 读写模式，在末尾添加内容，若不存在则创建
    "rb", "wb", "ab", "ab+"
    "a+b", "wb+", "w+b"
    "ab+", "a+b"
    二进制模式

     fopen() 返回一个文件指针，它并不指向实际的文件，而指向一个数据对象，其中包含操作文件的 IO 函数所用的缓冲区信息。
     在文件操作结束后，通过 int fclose(FILE *fp) 关闭文件。在较正式的程序中，需要显式检查其是否调用失败，磁盘已满、移动硬盘被移除和 IO 错误均会导致调用失败。

    (2). getc()、putc()

    类似与 getchar() 和 putchar()，不同之处在于需要指明文件。
    int getc(FILE *file), int putc(int ch, FILE *file)
    getc() 在读取时如果发现是文件结尾，会返回一个特殊值 EOF。 

    1
2
3
    while ((ch = getc(fp)) != EOF) { // determine the file is not empty
    putchar(ch);
}

    (3). fprintf()、fscanf()、fgets()、fputs()

    int fscanf(FILE *stream, const char *format, ...)
    int fprintf(FILE *stream, const char *format, ...)
    与 scanf() 和 printf() 类似，区别在于第一个参数是文件指针。

    char *fgets(char *buf, int count, FILE *file)
    指定存储位置和字符串大小，从文件中读取: 或读取到第一个换行符后面 (换行符也被读取)，或读到文件结尾，或读取指定的长度，读取完成后，添加一个空字符并存储在 buf 中。在遇到文件结尾时返回 NULL。

    int fputs(char *buf, FILE *file)
    把字符数组中的内容写入文件中。

    (4). fseek()、ftell()、rewind()

    int fseek(FILE *file, long offset, int origin)
    把文件视为字节序列，移动文件内的指针，第二个参数为移动的偏移量 (可以为正、负、0)，第三个参数为起始点，根据 ANSI 标准，在 <stdio.h> 定义了几个明示常量：

    mode of fseek()
    SEEK_SET 文件开始 SEEK_CUR 当前位置 SEEK_END 文件结尾

    旧的实现可能缺少这些定义，此时用 0L、1L、2L 表示。调用成功函数返回 0，出错则返回 -1。

    ftell(FILE *file)
    返回文件当前位置。需要注意，当以追加模式打开文件后，未进行任何 IO 操作前，这个时刻的内部指针仍指向文件头部。

    rewind(FILE *file)
    将文件内部的指针重新指向文件的开头。

    (5). fflush()

    int fflush(FILE *fp)
    C 标准库中所有 IO 都是缓冲 IO。此函数把输出缓冲区所有未写入到指定文件的数据写入，这个过程被称为刷新缓冲区。对输入缓冲区使用此函数时未定义的；任意读写模式下，只要最近一次操作不是输入，就可以用这个函数更新流。

    (6). feof()、ferror()

    读取到文件结尾或读取错误时，标准输入函数都会返回 EOF，这两个函数用于区分这两种情况。
    int feof(FILE *file),  int ferror(FILE *file)
    在上一次输入操作检测到文件结尾时，feof() 返回一个非 0 值，否则返回 0；当读或写遇到错误，ferror() 返回非 0，否则返回 0。
