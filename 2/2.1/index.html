    2. 引用形参和指针形参

     形参是函数内的临时变量，和一般的变量初始化一样，拷贝不会影响原值。已知引用也是通过指针实现的，当需要修改原对象，可以设置形参为引用或指针，二者区别不大。
     当不需要修改原对象，通过指针或引用传入可以避免对原对象进行拷贝 (部分对象甚至不支持拷贝)，节省一部分空间。此时最好使用常量引用，好处在于：① 可以直接传入对象，不必取引用；② 限制对原对象修改；③ 普通引用会限制 const 对象、字面量、需要类型转换的对象的传入。

    3. 可变参数

     为实现可变参数，C++11 提供了两种方法。若所有实参类型都相同，可使用 initializer_list 类型的形参，它定义在同名的头文件中；如果实参的类型不同，可以编写一个可变参数模板。在与 C 接口交互的程序中，一般使用省略符 ...，只对于 C 和 C++ 通用的类型使用它。

    initiallizer_list
    initializer_list<T> lst; ...
    lst.begin() lst.end()
    lst.size

    和 vector 不同的是，initializer_list 中的永远是常量。在传递实参的时候，要把参数写在花括号内。
